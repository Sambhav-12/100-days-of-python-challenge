# Day 10: Terminal Calculator & Functions with Outputs

A command-line calculator app built as part of the **#100-Days-Of-Python-Challenge**. This project focuses on **functions with outputs (`return` statement)**, **docstrings**, **dictionary function mapping**, and a little bit of **recursion**.

---

## 💡 Key Concepts Learned

- **Functions with Outputs (`return`):** Learned how `return` passes a value back to the caller (unlike `print()`, which only displays text), enabling chained math operations.
- **Docstrings (`"""..."""`):** Used to document functions directly below their declaration for hover-over help and auto-documentation.
- **First-Class Functions in Dictionaries:** Stored function references as dictionary values (`"+" : add`) to look up and execute operations dynamically based on user input.
- **While Loops & Recursion:** Combined boolean flags (`should_accumulate`) for continuous calculations with recursive function calls (`calculator()`) to reset state.
