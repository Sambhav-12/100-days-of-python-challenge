# Day 11: Blackjack CLI Game 🃏

An interactive Command Line Interface (CLI) implementation of the classic Blackjack game built with Python. This project focuses on state management, game loop mechanics, and dynamic list operations

## 🚀 Features
- **Dynamic Card Dealing:** Generates random cards for both the player and the dealer
- **State Tracking:** Handles player/dealer scores dynamically, including variable Ace values ($1$ or $11$).
- **Game Loops:** Manages turns using controlled loops for hit/stand mechanics and automatic dealer .
- **Game Reset:** Built-in loop to instantly restart a new round upon completion.

## 💡 Key Engineering Takeaways
- Utilized Boolean flags and nested `while` loops to control execution flow cleanly without infinite execution risks.
- Practiced state mutation through dynamic list operations (`.append()`, `sum()`)
