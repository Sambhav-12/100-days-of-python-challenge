# Day 1: Band Name Generator

## What I Built
A simple command-line program that asks the user for their city and pet's name, then combines them to generate a unique band name.

## Key Concepts Learned
* **Print Statements & String Manipulation:** Learning how to output text and use `\n` to print on new lines.
* **String Concatenation:** Joining different strings together using the `+` operator.
* **The `input()` Function:** Taking user input directly from the console.
* **Variables:** Storing user data in named containers to use later.

## Errors Faced & Solved
* **SyntaxError:** I accidentally missed a closing quote `"` in one of my print statements. I learned that Python needs exact syntax to compile.
* **NameError:** I mistyped a variable name when trying to print it. Python couldn't find where the variable was defined.
* **IndentationError:** I learned how we need to have proper spacing in python language.
