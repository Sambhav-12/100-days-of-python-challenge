# Day 9: Dictionaries, Nesting & Silent Auction Project

## 📌 Overview
Day 9 focused on understanding Python **Dictionaries** and **Nesting** data structures (lists inside dictionaries, dictionaries inside lists). The day culminated in building a command-line **Silent Auction Program** that dynamically collects user bids, clears the terminal screen between bidders, and calculates the highest bidder using key-value manipulation.

---

## 🧠 Key Concepts Mastered

### 1. Dictionaries vs. Lists (Key Lookup vs. Indexing)
- **Lists** are ordered collections accessed via **numerical indices** (e.g., `my_list[0]`).
- **Dictionaries** store data in **Key-Value pairs** and are accessed directly using their **keys** (e.g., `my_dict["key"]`).
- Attempting to access a dictionary using a numerical index results in a `KeyError` unless the integer itself happens to be a key.
